# AI Prompts Used

## Prompt 1 – HTML

Design a modern "24-Hour AI Hackathon" section for a developer conference website called DEVCONF 2026.

What it should have:

- Generate semantic HTML5 only (no CSS).
- Match a modern blue, white, and dark conference theme.
- Include:
  - Section heading
  - Description
  - Prize distribution cards
  - Statistics cards
  - Challenge tracks
  - Register button
- Keep the HTML beginner-friendly and well-indented.

---

## Prompt 2 – CSS

Generate modern CSS for the Hackathon section.

What it should have:

- Match the DEVCONF 2026 design.
- Use CSS Grid.
- Add rounded corners, hover effects, soft shadows, and responsive spacing.
- Keep the CSS beginner-friendly with comments.
