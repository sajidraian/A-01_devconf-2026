# DEVCONF 2026

A modern, responsive landing page for a fictional developer conference built using HTML and CSS.

## Live Website

https://sajidraian.github.io/A-01_devconf-2026/

## GitHub Repository

https://github.com/sajidraian/A-01_devconf-2026

---

## Features

- Responsive navigation bar
- Hero section with call-to-action
- Speaker showcase section
- Pricing section with three plans
- AI-generated Hackathon Challenge section
- Modern footer with social media links
- Hover effects and smooth card animations
- Clean and responsive layout using Flexbox and CSS Grid

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Font Awesome Icons

---

## AI Generated Section

The **24-Hour AI Hackathon** section was created with the help of ChatGPT as required in the assignment.

The AI was used to:
- Generate the HTML structure
- Generate the CSS styling
- Improve the overall design while matching the existing website theme
